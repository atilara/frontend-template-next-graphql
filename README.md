# Instructions

Starter template for [NextJS](https://nextjs.org/).

## Features

- GraphQL client w/ [apollo](https://github.com/apollographql/apollo-client)
- GraphQL Code Generator w/ [codegen](https://github.com/dotansimha/graphql-code-generator)
- Tests w/ [jest](https://github.com/facebook/jest)
- Cookies w/ [nookies](https://github.com/maticzav/nookies)
